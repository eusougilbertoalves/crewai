# Testes com EvolutionAPI

## uv venv
### Create
uv venv

### Activate
source .venv/bin/activate

### Deactivate
deactivate

## Init Project
uv init

## Install Requirements
uv pip install -r requirements.txt

## Run
uv run evo_msgsender_test.py
streamlit run st_evo_msgsender.py
